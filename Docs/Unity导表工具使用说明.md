# 🎮 Unity导表工具 - 快速使用说明

## ✨ 新功能！Unity内置导表工具

现在你可以直接在Unity编辑器中转换配置表，无需运行外部脚本！

---

## 🎯 在哪里找到？

### Unity编辑器顶部菜单栏：

```
┌─────────────────────────────────────────┐
│ File  Edit  Assets  GameObject  Tools   │ ← 点击这里
└─────────────────────────────────────────┘
```

点击 **Tools** 后，你会看到：

```
Tools
  │
  ├─ 导表 (CSV → JSON)          ⭐ 点击这个转换配置
  │
  ├─ 打开配置文件夹              📁 打开CSV文件夹
  │
  └─ 打开Unity配置文件夹         📁 打开JSON文件夹
```

---

## 🚀 三步快速使用

### 第1步：编辑CSV配置

**点击 Unity菜单：**
```
Tools → 打开配置文件夹
```

**用Excel打开并编辑：**
- `cards.csv` - 卡牌数据
- `decks.csv` - 卡组配置
- `gameConfig.csv` - 游戏设置

**保存文件** (Ctrl+S / Cmd+S)

---

### 第2步：运行导表工具

**点击 Unity菜单：**
```
Tools → 导表 (CSV → JSON)
```

**等待1秒...**

**看到弹窗：**
```
┌──────────────────────────────┐
│         导表成功             │
│                              │
│  配置表转换完成！            │
│                              │
│  ✅ 卡牌: 12 张              │
│  ✅ 卡组: 2 个               │
│  ✅ 游戏配置: 已更新         │
│                              │
│  文件已保存到:               │
│  Assets/Resources/Config     │
│                              │
│         [ 确定 ]             │
└──────────────────────────────┘
```

**点击"确定"**

---

### 第3步：测试游戏

**运行游戏场景**

**配置立即生效！** ✨

## 💡 实际使用示例

### 场景：调整"士兵"卡的攻击力

**传统方式：**
```
1. 打开CSV文件
2. 修改数据
3. 保存
4. 切换到终端
5. 运行脚本
6. 切换回Unity
7. 等待刷新
⏱️ 2-3分钟
```

**使用Unity工具：**
```
1. Tools → 打开配置文件夹
2. Excel编辑 cards.csv
3. 修改 soldier 的 attack: 3 → 4
4. 保存 (Ctrl+S)
5. Tools → 导表
6. 完成！
⏱️ 30秒
```

**效率提升：4倍！**

## 📺 使用演示

### 示例1：添加新卡牌

**1. 点击菜单打开配置文件夹**
```
Tools → 打开配置文件夹
```

**2. Excel编辑 cards.csv，添加新行：**
```
knight,骑士,骑士动画,2,4,4,Character,Normal,player,重装骑士
```

**3. Excel编辑 decks.csv，添加新行：**
```
playerStarterDeck,圣骑士起始卡组,knight,2,骑士
```

**4. 保存所有文件**

**5. Unity菜单导表**
```
Tools → 导表 (CSV → JSON)
```

**6. 看到成功弹窗，点击确定**

**7. 运行游戏，新卡牌已添加！**

---

### 示例2：批量调整平衡性

**需求：** 所有角色卡攻击力+1

**1. Tools → 打开配置文件夹**

**2. Excel打开 cards.csv**

**3. 使用Excel筛选：**
- 点击表头 → 数据 → 筛选
- cardType 列 → 只选 "Character"

**4. 批量修改 attack 列：**
- 选中所有角色卡的攻击力
- 使用公式或手动+1

**5. 保存文件**

**6. Tools → 导表**

**7. 完成！所有角色卡已增强**

## ⚠️ 常见问题

### Q: 菜单中没有Tools选项？

**A: 检查Editor脚本**
```
1. 确认文件存在：
   Assets/Editor/ConfigTableConverter.cs

2. Unity菜单 → Assets → Refresh (Ctrl+R)

3. 等待Unity编译完成

4. 如仍未出现，重启Unity
```

---

### Q: 点击"导表"无反应？

**A: 检查CSV文件**
```
1. 确认Config文件夹存在

2. 确认CSV文件存在：
   - cards.csv
   - decks.csv
   - gameConfig.csv

3. 查看Unity Console的错误信息
```

---

### Q: 转换后游戏中没生效？

**A: 刷新Unity**
```
1. Unity菜单 → Assets → Refresh

2. 重新运行游戏场景

3. 确认JSON文件已更新：
   Assets/Resources/Config/
```
